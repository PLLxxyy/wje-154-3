# 手工作坊 - 手工作品展示平台

一个纯前端的手工作品展示平台，数据存储在浏览器 localStorage 中，无需后端服务。

## 技术栈

- **构建工具**: Vite 5
- **前端框架**: React 18
- **语言**: TypeScript
- **数据持久化**: localStorage
- **样式**: 内联 CSS（index.html style 标签）

## 启动方式

```bash
npm install
npm run dev
```

启动后访问 http://localhost:5173

## 功能说明

### 瀑布流首页
- 作品以瀑布流布局展示，每张卡片包含封面、标题、作者头像和点赞数
- 顶部搜索框支持按标题或材料关键词搜索
- 分类筛选栏：烘焙、编织、木工、陶艺、皮具、花艺
- 支持按最新和最热排序

### 作品详情
- 完整的图文教程，分步骤展示，每步有图片和文字说明
- 右侧显示材料清单、难度等级、小贴士
- 点赞、收藏功能
- 评论区（登录后可发评论）
- 作者信息卡片，显示关注数和作品数，支持关注/取关

### 发布作品
- 分步填写：先写标题、选分类、设难度
- 逐步添加教程内容，每步配图和说明
- 填写材料清单和小贴士
- 提交后作品出现在首页和个人主页

### 个人主页
- "我的作品"和"我的收藏"两个 Tab
- 我的作品支持查看、上架/下架操作
- 我的收藏展示已收藏的作品

### 用户系统
- 简易登录（输入昵称即可）
- 登录后可点赞、收藏、评论、发布作品、关注作者
- 退出登录

## 预置数据

内置 13 个手工作品，覆盖 6 个分类：
- 烘焙：法式可颂、日式轻芝士蛋糕、伯爵红茶曲奇
- 编织：冰条线手提包、渐变色围巾
- 木工：原木砧板、榫卯小板凳
- 陶艺：手捏咖啡杯、侘寂风花瓶
- 皮具：植鞣革短夹、钥匙扣挂件
- 花艺：韩式花束包装、永生花相框
